## Set up for Unix, MacOS
```bash
virtualenv env
source env/bin/activate
pip3 install -r requirements.txt
```
## Set up for Windows
```bash
virtualenv env
.\env\Scripts\activate
pip3 install -r requirements.txt
```
## Set up database
```bash
python manage.py makemigrations
python manage.py migrate
```
## Start the app
```bash
python manage.py runserver
```
